Hi there! I'm Tom.

So… I know a thing or two about web development. It all started back in Deerfield Beach High School with two Web Design courses. They had us use Dreamweaver and Fireworks CS3 to make front end user interfaces. We didn't do much in terms of programming but it did catch my attention. Not too long after graduating, I started to look into PHP and MySQL to make a message board! Followed copious tutorials from an English lad in phpacademy (now CodeCourse). Learned how to connect to backends, manage user accounts, look for vulnerabilities and store sensitive information securely. His videos were invaluable.

Fast-forward to today, I've been keeping up with the latest trends and doing a lot of self-studying. CSS flexbox and grid are game changers! Dabbled a little with Node.js, Angular2+, React, and many more platforms & libraries.
